# SGC-MEDICO
Sistema de Gestión de Citas Médicas - Proyecto de título Analista Programador 2025.  Aplicación web que digitaliza el agendamiento de citas médicas con control automático de inasistencias, gestión de disponibilidad de profesionales y notificaciones automatizadas.  Stack: Django REST + React + Tailwind CSS Roles: Paciente, Profesional, AdminAuth:JWT
